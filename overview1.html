    Features

    Professional Search Page.

    Designed for finding other TAMVOES users and organizations.

    TAMVOES is...

    A health management platform that empowers users by granting access to personalized health information, fostering informed healthcare decisions.

    Why Redesign?

    As TAMVOES migrated to React, it became clear that enhancing the visual aesthetics of the existing user interface was essential to align with the overall design of the updated React-based website.

    Search Page Before

    Search Page After

    Process

    1. Brainstorming: Discussed the direction of the redesign with my manager, ultimately reaching a conclusion that the page could benefit from a more contemporary aesthetic.
    2. Mockup: Developed multiple mockups aligned with the overall aesthetic of the website, presented them to the team, and I iteratively made changes to arrive at the final design.
    3. Developement: The mockup was then turned into a fully functional page with the reference of the legacy code, Bootstrap, and Formik. In the process, I also created several API classes that retrieve client and organization information.
    Result: Currently deployed at the TAMVOES website (Log in/Register > Professional Search). The revamped professional search page presents a contemporary aesthetic along with a new view toggle, providing users with options for both an expanded and compact view. Additionally, it now incorporates an advanced search feature enabling users to search by location, website, or phone number.
